# Growth

Umbrella entry point for go-to-market and growth work. Invoke this, then load the member skills below by relevance to the task instead of pulling each skill one by one.

## When to use

- Shaping a landing page, pricing page, or positioning end-to-end
- Any task that spans persuasive copy, pricing, and discoverability

This covers copy, pricing, and discoverability; for designing and building the page UI itself, pull `build-web-apps`.

For a single narrow task, the specific member skill below still applies on its own.

## Member skills — load these

| Skill | Pull in when |
|---|---|
| `copywriting` | Writing or rewriting persuasive copy: landing pages, headlines, CTAs, value props, positioning |
| `pricing-strategy` | Packaging, tiers, freemium, trials, value metrics, willingness to pay, pricing pages |
| `seo-audit` | Auditing or diagnosing technical and on-page SEO, rankings, traffic drops, Core Web Vitals |
| `revenue-centric-design` | Conversion/CRO, onboarding & activation, churn/retention, pricing psychology, positioning/ICP — advisory playbook (excludes gambling/betting products) |

## How to use

1. Invoke the member skills relevant to the task; pull the copy, pricing, and SEO members together when shaping a full marketing page, and add `revenue-centric-design` for conversion, activation, or retention concerns.
2. Typical flow: `pricing-strategy` (what to sell and how to package) → `copywriting` (how to say it) → `seo-audit` (make sure it can be found).
3. Follow each member skill's own guidance; they are the source of truth.

## Note

This is a dispatcher. It holds no rules of its own — the authority lives in the member skills. Keep them as the source of truth and update them, not this file.
